## Como rodar localmente

```bash
npm install
npm run dev
```

## Deploy (Netlify)

1. Push para GitHub
2. Conectar repositório no Netlify
3. Build command: `npm run build`
4. Publish directory: `dist`
5. Apontar domínio `fjsmart.com.br` nas DNS settings
